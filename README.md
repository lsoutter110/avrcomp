# avrcomp

I reccomend you add avrcomp to the system path so it can be run anywhere.

### Run
```avrcomp filename [options]```

### Options
```-p [compilation processor] [upload processor]```
```-c [programmer]```
```-f [frequency]```
```-l [filename list]```

### Example
```avrcomp mpu6050 -l uart_wrapper twi_wrapper```
